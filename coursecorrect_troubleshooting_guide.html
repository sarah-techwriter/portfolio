Troubleshooting Guide for CourseCorrect

This guide provides solutions to the most common issues users encounter when using the CourseCorrect platform. Before reaching out to support, please follow the steps below to resolve your issue quickly.

Account Issues

Login Problems

Symptoms Possible Cause Solution Prevention
Google sign-in button not working, authentication fails Personal Google account used, pop-up blocked, multiple accounts 1. Ensure using business/educational Google account
2. Enable pop-ups
3. Clear browser cache		 Use business Google account, whitelist CourseCorrect domain
"Account type not supported" message Using personal Gmail account Switch to business/educational Google account Verify account type before attempting login
Stuck after clicking "continue" Process freezes after clicking "continue" on the permissions screen Declinced access permissions, firewall blocking 1. Re-authorize CourseCorrect access
2. Accept name.email permissions
3. Check corporate firewall settings
